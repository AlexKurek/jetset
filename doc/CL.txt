3) run step A or B
    A)run all notebooks and build rst file with script
        ./build_rst_files.sh -e documentation_notebooks #run all notebooks found in documentation_notebooks
        OR
        ./build_rst_files.sh -e documentation_notebooks/dir-name #runs only specific sub-directory


    B) or ONLY BUILD RST
        ./build_rst_files.sh documentation_notebooks
        OR
        ./build_rst_files.sh  documentation_notebooks/dir-name



4)update the rst in user_guide from the notebooks in documentation_notebooks
  find documentation_notebooks -name '*.rst' | cpio -pdm ./user_guide
  find documentation_notebooks -name '*.png' | cpio -pdm ./user_guide

5)builds the docs
  sphinx-build -b html ./ build
