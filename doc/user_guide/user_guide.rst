=================
JetSeT User Guide
=================

The following tutorial covers a large fraction of the code features. Please, follow the order
reported in the table below. A quick start is available on the github repository `here <https://github.com/andreatramacere/jetset/tree/1.1.2/notebooks>`_,
but before I suggest to through the tutorials below.You can also run the quick start notebook on binder, by clicking the badge below:

.. image:: https://mybinder.org/badge_logo.svg
 :target: https://mybinder.org/v2/gh/andreatramacere/jetset/master

.. note::
   Items with the  *New (v1.2.0)*  flag refer to new api or improved fetures for  v1.2.0, and it is warmly advised to read the section also for experienced users
